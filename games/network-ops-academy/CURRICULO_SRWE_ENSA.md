# Network Ops Academy — Cobertura curricular SRWE + ENSA

Esta versão expande o jogo para cobrir os grandes blocos práticos dos cursos Cisco SRWE e ENSA.

## SRWE
- VLANs e portas access — NOA-101
- Trunks 802.1Q e native VLAN — NOA-102
- Inter-VLAN Routing / Router-on-a-Stick — NOA-103
- STP/RSTP e root bridge — NOA-104
- EtherChannel / LACP — NOA-105
- Port Security — NOA-106
- WLAN / WLC / SSID / WPA2-PSK — NOA-107
- FHRP / HSRP — NOA-108
- IPv6 routing e rota estática — NOA-109
- SLAAC / Router Advertisement / EUI-64 — NOA-110

## ENSA
- OSPF single-area — NOA-201
- NAT/PAT — NOA-202
- ACL standard/extended e wildcard — NOA-203
- DHCP Relay / ip helper-address — NOA-204
- Troubleshooting WAN — NOA-205
- NTP, Syslog e SNMP — NOA-206
- QoS para voz — NOA-207
- VPN IPsec site-to-site — NOA-208
- Automação com Ansible, JSON e infra-as-code — NOA-209
- OSPF multiárea / ABR / área 0 e área 1 — NOA-210
- SSH e hardening de gestão remota — NOA-211
- SDN, REST API, JSON e intent-based networking — NOA-212

## Integração final
- Incidente crítico multicamada — NOA-300

Observação: a cobertura é educacional/simulada. O objetivo é consolidar conceitos, comandos e troubleshooting no navegador; não substitui Packet Tracer ou hardware real para detalhes específicos de plataforma.
