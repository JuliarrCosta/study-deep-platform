Como usar localmente

1. Baixe ou clone o repositório.
2. Abra index.html no navegador.
3. Entre em um módulo e estude por aulas, questões e revisão.
4. O progresso fica salvo no navegador via localStorage.

Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie todos os arquivos deste pacote.
3. Vá em Settings → Pages.
4. Em Deploy from branch, escolha main e pasta /root.
5. Acesse a URL gerada pelo GitHub Pages.

Para futuros contribuidores

Mantenha a estrutura index.html + pages/. Ao adicionar um novo módulo, crie uma nova página HTML dentro de pages/, use a barra superior padrão e adicione o card correspondente nesta página principal. Evite dependências externas para manter a plataforma offline e fácil de publicar.
